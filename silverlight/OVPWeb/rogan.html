For the best experience, Microsoft recommends the following minimum system requirements:
   - Video is supported on following browsers:
       - Internet Explorer 6, 7 for Windows (2003, XP SP2 or greater, and Vista)
       - Firefox 1.5, 2, 3 on Windows (2003, XP SP2 or greater, and Vista)
       - Safari 2, 3 for Macintosh (Intel only)
